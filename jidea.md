## Shortcuts

### Build & Run
    cmd + shift + F9		Compile
    shift + F9				Debug
    cmd + F9				Make Project
    shift + F10				Run

### Debugging
	F9						Resume
	F8						Step Over
	F7						Step Into
	shift + F8				Step Out
	cmd + F2				Stop
	shift + cmd + F8		View Breakpoints
