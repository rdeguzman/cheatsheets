### Layout Split Vertically
	⌥ + ⌘ + 1
	⌥ + ⌘ + 2 # split to two columns

### Move file to other column
    ⌃ + ⇧ + 1 # move to 1st column
    ⌃ + ⇧ + 2 # move to 2nd column

### Find File
	 ⌘ + P
